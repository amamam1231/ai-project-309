## 🚀 Технологии

- HTML/CSS/JavaScript (для базовой структуры и стилей)  
- React/Next.js (для динамических элементов, таких как онлайн-запись и система лояльности)  
- Tailwind CSS (для быстрого и гибкого стилевого оформления в винтажном стиле)  
- Framer Motion (для анимаций и интерактивности)  
- Firebase (для хранения данных о клиентах и системе лояльности)
